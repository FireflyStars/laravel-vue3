## How to set up the project?
- `SANCTUM_STATEFUL_DOMAINS=localhost:8000` add this code to .env file
- composer install
- nmp install Or yarn
- cp .env.example .env
- php artisan db:create laravel_vue
- php artisan migrate
- php artisan db:seed
- php artisan serve
